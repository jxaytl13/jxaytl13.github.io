Hierarchy Vision Plus

介绍

Hierarchy Vision Plus 是一个强大的 Unity 编辑器扩展包，增强了 Unity 中的层级和项目视图。它为开发者提供了高效的场景对象和项目资源管理工具。

系统需求

  • Unity 2018.3+
  • 已测试版本：
    • Unity 2018.3 LTS
    • Unity 2019.3 LTS
    • Unity 2020.3 LTS
    • Unity 2021.3 LTS
    • Unity 2022.3 LTS
  • 兼容 Windows/MacOS/Linux

安装

  1. 在 Unity 编辑器中，导航至 Assets > Import Package > Custom Package
  2. 选择下载的 EditorExtension.unitypackage 文件
  3. 确保所有文件都已选中，然后点击"导入"

主要功能

1. 隔离显示

  • ~ : 切换到层级隔离模式（仅显示选定节点）
  • Ctrl + ~ : 切换到场景隔离模式（仅在层级和场景窗口显示选定节点）

2. 材质预览和编辑

直接在层级中预览和编辑材质名称

  • 操作方法：
    • 左键点击材质颜色块：定位材质
    • Ctrl + 悬停在材质颜色块上：预览材质
    • 双击材质颜色块：编辑材质名称

3. 节点图标

  • 节点组件图标显示
  • 快捷键：Ctrl + Shift + ~ 切换节点图标和组件图标

4. 背景颜色编辑

  • 支持预设和自定义颜色
  • 可以选择是否应用到子节点
  • 按 Alt + 点击节点图标：更改节点背景颜色

5. 树形视图

  • 显示对象层级，清晰的视觉层次呈现

6. 文件夹颜色管理

  • 支持预设和自定义颜色
  • 可以选择是否应用到子文件夹
  • 按 Alt + 点击文件夹图标：更改文件夹背景颜色

7. 文件夹树结构

  • 清晰显示文件夹层级

版本信息

  • 支持 Unity 2018.3.0+

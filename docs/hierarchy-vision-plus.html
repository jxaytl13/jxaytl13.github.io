Hierarchy Vision Plus

Introduction

Hierarchy Vision Plus is a powerful Unity editor extension package that enhances both the Hierarchy and Project views in Unity. It provides developers with advanced tools for efficient scene object and project resource management.

System Requirements

  • Unity 2018.3+
  • Tested on:
    • Unity 2018.3 LTS
    • Unity 2019.3 LTS
    • Unity 2020.3 LTS
    • Unity 2021.3 LTS
    • Unity 2022.3 LTS
  • Compatible with Windows/MacOS/Linux

Installation

  1. In Unity Editor, navigate to Assets > Import Package > Custom Package
  2. Select the downloaded EditorExtension.unitypackage file
  3. Ensure all files are selected and click "Import"

Main Features

1. Quarantine Display

  • Shortcut key:
    • ~ : Switch to Hierarchy Isolation Mode (only display selected nodes)
    • Ctrl + ~ : Switch to Scene isolation mode (Hierarchy and Scene windows only display selected nodes)

2. Material Preview and Editing

Directly preview and edit material names in Hierarchy

  • Operation method:
    • Left-click on the material color block: locate the material
    • Ctrl + Hover over material color block: preview material
    • Double-click the material color block: Edit material name

3. Node icon

  • Node component icon display
  • Hotkey: Ctrl + Shift + ~ to switch between node icons and component icons

4. Background color editing

  • Support preset and custom colors
  • You can choose whether to apply to child nodes
  • Press Alt + click the node icon to change the node background color

5. Tree View

  • Display object hierarchy, a clear visual hierarchy presentation.

6. Folder Color Management

  • Support preset and custom colors
  • You can choose whether to apply to subfolders
  • Press Alt + click the folder icon: change folder background color

7. Folder tree structure

  • Clearly display the folder hierarchy

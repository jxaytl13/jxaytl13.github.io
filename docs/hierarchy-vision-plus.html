Hierarchy Vision Plus

Overview

Hierarchy Vision Plus is a powerful Unity editor extension that enhances your scene hierarchy management by displaying component icons next to GameObjects. This visual aid makes it easier to identify and organize objects in your scenes.

Features

  • Component icon display in hierarchy
  • Easy hotkey toggle functionality
  • Customizable icon settings
  • Performance optimized
  • Unity 2019.4 and above support

Installation

  1. Open Unity Package Manager
  2. Click the + button in the top-left corner
  3. Select "Add package from git URL"
  4. Enter the package URL
  5. Click Add

Usage

After installation, the component icons will automatically appear in your hierarchy window. You can toggle the visibility using the following hotkey:

  • Windows: Ctrl + Shift + H
  • macOS: Cmd + Shift + H

Configuration

You can customize the extension through the preferences window:

  1. Go to Edit > Preferences
  2. Select Hierarchy Vision Plus tab
  3. Adjust settings to your needs

UI列表动画器

介绍

UI列表动画器（UIListAnimation）是一个强大且灵活的Unity UI列表动画插件，可以轻松为列表或网格中的UI元素添加动画效果。该插件依赖于DOTween插件。

系统需求

  • Unity 2020.3+（未测试较旧的2020版本，可能兼容）
  • DOTween/Pro（必需）

安装步骤

  1. 在Unity编辑器中，选择 资源 > 导入包 > 自定义包
  2. 选择下载的 EditorExtension.unitypackage 文件
  3. 确保已安装DOTween/Pro插件

主要功能

支持的动画类型

  • 位置
  • 缩放
  • 旋转
  • 画布组

播放模式

  • 普通
  • 网格

动画配置选项

  • 自定义缓动函数（多种内置缓动类型）
  • 自定义动画曲线
  • 灵活的时间控制（间隔模式/持续时间模式）
  • 可调整的起始延迟
  • 支持动画预览（在运行状态）

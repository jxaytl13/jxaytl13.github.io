UI List Animator

Introduction

UI List Animation (UIListAnimation) is a powerful and flexible Unity UI list animation plugin that can easily add animation effects to UI elements in lists or grids. This plugin relies on the DOTween plugin.

System Requirements

  • Unity 2020.3+ (older 2020 versions not tested, may be compatible)
  • DOTween/Pro（Required）

Installation Steps

  1. In the Unity editor, select Assets > Import Package > Custom Package
  2. Select the downloaded EditorExtension.unitypackage file
  3. Ensure that the DOTween/Pro plugin is installed.

Main Functions

Supported Animation Types

  • Position
  • Scale
  • Rotation
  • CanvasGroup

Play Mode

  • Normal
  • Grid

Animation Configuration Options

  • Custom easing function (multiple built-in Ease types)
  • Custom animation curve
  • Flexible time control (Interval mode/Duration mode)
  • Adjustable start delay
  • Support animation preview (in running state)

UI List Animator

Overview

UI List Animator is a versatile Unity package that brings life to your UI elements with smooth, professional-grade animations. Perfect for game menus, item lists, and any UI that needs dynamic transitions.

Features

  • Pre-built animation presets
  • Customizable animation parameters
  • Event-driven animation system
  • Performance optimized
  • Easy integration with existing UI

Installation

  1. Open Unity Package Manager
  2. Click the + button in the top-left corner
  3. Select "Add package from git URL"
  4. Enter the package URL
  5. Click Add

Quick Start

To add animations to your UI list:

  1. Add UIListAnimator component to your list container
  2. Configure animation settings in the inspector
  3. Call Animate() method to trigger animations

Animation Types

  • Fade In/Out
  • Scale Up/Down
  • Slide In/Out
  • Bounce
  • Custom animations
